Player Object -> window.Player()

A Player object, returned by the Player() constructor, represents a player. It may be changed in the future, but for the moment some mechanics of Chess++ are made considering that there are only two players, stored in the windows.joueur array, thus this constructor should not be used.
In p5.setup() >
joueur = [new Joueur("blanc","Gilbert"), new Joueur("noir","Patrick")]; //crée les deux joueurs de base
